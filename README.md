# Mapolis

Educational geography game. Phase B web beta.

## Structure

- `play/` — game (deploys to play.mapolis.app)
- `admin/` — admin (deploys to admin.mapolis.app, currently a stub)
- `assets/` — shared JS, lands in Step 2
- `docs/` — blueprints and handoffs

## Deploy

Both subdirectories deploy to separate Netlify sites on push to `main`.
